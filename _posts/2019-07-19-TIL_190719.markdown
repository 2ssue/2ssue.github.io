* javascript 연산 방법
    - `!(num)`: `Number` 형태를 `boolean`형태로 바꿔준다.
        + !(3) = true !(0) = false
    - `+(bool)`: `boolean`형태를 `Number`형태로 바꿔준다.
        + +(true) = 1 +(false) = 0 
    - `**`: 거듭제곱
        + 2**3 = 8

* javascript `apply()` 함수
