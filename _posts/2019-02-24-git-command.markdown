git 명령어 참고

### git remote add {name}

{name} 이름을 가진 remote 저장소를 추가한다.

### git add .

수정된 전부(.)를 현재 remote에 추가한다.

### git commit -m "message"

message를 add된 목록들에 commit 한다.

### git push -u origin master

origin에 추가된 내용을 master에 push한다.
